Project Outline

Meteor Game Engine

Overview:

Meteor is a custom Game Engine that I am working on, Ideally it is supposed to work on Windows. As of now there is no goal but I thought working on an engine in my free time would help me better understand game development.

Roles:

Th engine is a group project and has been written by me and a friend in our free time, I was responsible for setting up all the libraries in the project.

Learnings:

The engine has been very helpful in understanding the aspects of game development and rendering overall. I have learnt so much about some core engine concepts such as enteties,components,cameras, profilers, events etc. Since I had to write my own renderer and shader it has helped me get a better understanding of custom shaders as well.
