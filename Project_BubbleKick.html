Project Outline

Bubble Kick

Overview:

Bubble Kick is a game that I worked on and submitted for the Voodoo 2021 Fall Competetion. The theme of the competetion was balls. The objective of the game is to use your gun to shoot balls at the zombies. The balls stick to the enemies and once an enemy is covered in balls, the balls combine to form a bubble which immobilizes the enemy. The player can then kick those enemies into the water.

Roles:

All levels for the game were designed by me. I wrote the code for the entire game, including a modular level generator which takes a 25x25 pixel image and converts that into a level map with colors assigned for enemies, spawn points, bridges etc. The gun used was modelled by me in Blender and I also wrote the object pooling system for the bullets.

The project includes the following external SDKs which were integrated by me

  • Voodoo Analytics
  • Facebook Analytics
  • GameAnalytics

Learnings:

The project was a great learning experience as I got to write my own level map generator and I got to work with multiple AI enemy agents. The project also helped me familiarise myself with object pooling in Unity3D.
