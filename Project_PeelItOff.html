Project Outline

Peel It Off

Overview:

Peel It Off is a trivia project that I worked on. The objective is to guess the image by peeling strips from the canvas. The player and opponent get to remove one strip at a time and then they can take a guess at what's hidden behind the strips. A right guess will let you win but too many incorrect tries and you'll become shark food !

Roles:

The project required me to do multiple things such as

  • Programming
  • Level Design
  • Game Art
  • Third Party SDK integration

Learnings:

The project required me to handle user keyboard input and parse it accordingly. The keyboard on screen was made by me and helped me learn about UI constraints and layout settings. The peeling strips are 3D meshes that are manipulated at runtime. A 2D implementation did not seem appealing so I decided to learn how to modify 3D shapes instead to create a more realistic peel effect. I also had to place checks so that the game would reward the player even if their spelling was off by a few alphabets so learning string utilities came in very handy.
