Project Outline

Doodle Golf

Overview:

Doodle golf is my take on golf games combined with the popular drawing mechanic in hypercasual games. The game lets players explore their creativity by creating a golf map on the fly the goal is simple, to get the ball to land in the hole in one shot and while the board is empty the player is allowed to draw walls with their fingers through which they can guide the wall. There are many complicated mechanics in the game such as magnets, fans, portals, keys, etc.

Roles:

  • Created all Game out like the golf ball, the hole, the fans, magnets, keys, etc using blender and Adobe Photoshop.
  • wrote the game code to handle bowl movement, wall creation and game management. Also rude code for saving user data and session details rode the game code to handle ball movement, wall creation and game management. Also route code for saving user data and session details
  • designed over 150 levels for the game out of which 80 made the cut in the final version
  • wrote a custom level builder to allow me to create levels for the game using touch input on my phone

Learnings:

Doodle golf allowed me to learn the unity 2D physics system. Most of my unity experience has been with 3D physics but unity 2D physics was something which I learnt from this project. The fact that keyboard and mouse input is not precise enough to create levels forced me to write a custom level editor which I could run on my phone to create levels for the game with my fingers to give a more natural feel to the game. I think creating the level designing tool has helped me bolster my programming skills, I designed over 150 levels for the game and 80 of them made the cut. I learned that making easy levels is more difficult than making hard levels. Overall the project has been a great learning experience.
