Project Outline

Bubble Gum Run!

Overview:

Bubble Gum Run! was the first unity project I made under my independent studio DeepMind Interactive. The game is runner game about a girl who is trying to dodge obstacles while trying to collect jars of bubblegum to increase the size of tbe bubble. The player can use the bubble to fly over obstalces and to get a multiplier score boost at the end. 

Roles:

I wrote the code for the entire game which includes the code for

  • Flying
  • Running and player movement
  • Pickup and Obstacle Spawnning

I also integrated multiple analytics and ad network SDKs including but not limited to

  • Voodoo Analytics
  • Facebook Analytics
  • GameAnalytics
  • MoPub
  • AdMob
  • Google Ads

Learnings:

I learnt a lot from this project including things like setting up player movement, Unity Rigidbody physics, handling mobile imput and a few other unity basics.
