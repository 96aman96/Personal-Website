Project Outline

Card Run 3D

Overview:

Card Run 3D is a card based runner where the player must collect cards along the way and use them to overcome obstacles. It is one of my favourite projects as it originates from my obsession with card games like Hearthstone, Clash Royale, Card of Darkness to name a few. The game is my attempt to turn card games into simple, casual experiences. The player can pick up cards and drag them to grant abilities to the crowd that is running on the path. A bridge card can create bridges to cross gaps, A giant card can turn the player into a giant so that they make break walls and so on.

Roles:

All art (both 3D and 2D) was made by me including the cards, the 3D Models for the scene and the toon shader which is used for most materials. I also wrote the code to spawn the right card for the pcik up and the code which determines how each card affects the characters. All behaviours for the enemies and obstacles were written by me as well

I also integrated multiple analytics and ad network SDKs including but not limited to

  • Voodoo Analytics
  • Facebook Analytics
  • GameAnalytics
  • MoPub
  • AdMob
  • Google Ads

Learnings:

The project was especially challenging because unlike traditional card games,in casual games you need to make sure that the player has at least one solution in their hand at any given point. However you also need to make sure that there are not duplicate cards in the players hand. I had to write a specific function so that every pick up along the path would always check the obstacle ahead and make sure that the player had the solution to that obstacle. I also learnt a lot about random generation, weighted randoms, runtime instantiation among other things.
