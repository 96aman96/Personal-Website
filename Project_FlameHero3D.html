Project Outline

Flame Hero 3D

Overview:

Flame Hero 3D is a hyper casual project which I worked on which allows the player to lob fireballs at hordes of goons. The player can tap to shoot fireballs and use flamethrowers located in the palm of the players hands. A charge meter fills up as you take out enemies. Once full the player can unleash a tornado that will kill all enemies in an area.

Roles:

My work on the project involved numerous things. Like all others, this is a solo project so I had to use some 3D models from the Unity Asset Store. All programming was done by me

  • Setting up the player projectiles

  • Configuring enemies to move towards the player and use their weapons to attack player.

  • Creating all UI elements and writing code for them to control the game.

I also integrated multiple analytics and ad network SDKs including but not limited to

  • Voodoo Analytics
  • Facebook Analytics
  • GameAnalytics
  • MoPub
  • AdMob
  • Google Ads

Learnings:

The project helped me learn about CineMachine, projectiles and their motion, using NavMesh Agents and working with unity Particle Systems.
